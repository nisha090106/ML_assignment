Assignment 1

What is implemented
- Includes two notebooks: `Assignment 1.ipynb` and `operations.ipynb`.
- Covers dataset operations, data loading, cleaning, transformation, and exploratory analysis using Python.
- Demonstrates common Pandas and NumPy workflows for practical machine learning exercises.

How to run in Google Colab
1. Open [Google Colab](https://colab.research.google.com).
2. Upload `Assignment 1.ipynb` or `operations.ipynb` using File > Upload notebook.
3. If any dataset file is required, upload it to the Colab session or mount Google Drive.
4. Run the notebook cells sequentially from top to bottom.

Implementation notes
- The notebooks use Python data science libraries such as `pandas`, `numpy`, and `matplotlib`.
- If any import is missing in Colab, install it with `!pip install pandas numpy matplotlib`.
- The notebooks are self-contained and intended for step-by-step execution.

How to run
1. Open [Google Colab](https://colab.research.google.com).
2. Upload the notebook file from the desired assignment folder.
3. Upload any required dataset files from the same assignment folder, or mount Google Drive to access them.
4. Run the notebook cells sequentially from top to bottom.

Notes
- Most assignments include a `README.md` inside the assignment folder with details on what is implemented and how to run the notebook.
- If a notebook uses external Python packages, install them in Colab using `!pip install <package>`.
- Data files such as `csv` files are located in the assignment folders that require them.
- The repository is organized by assignment number for easy navigation.
